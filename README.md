# React Task Manager

A modern and responsive **Task Manager (To-Do List)** application built with **React** and **Vite**. This project allows users to add, complete, and delete tasks with a clean and user-friendly interface. Tasks are automatically saved in the browser using **Local Storage**, so they remain available even after refreshing the page.

---

## 📌 Project Overview

This project was developed as part of the **Vortex Tech Web Development Internship - Week 3 Assignment**. The main objective was to practice React fundamentals, component-based architecture, state management using the `useState` hook, and handling user interactions.

---

## ✨ Features

- ➕ Add new tasks
- ✅ Mark tasks as completed
- 🗑️ Delete completed tasks
- 💾 Save tasks in Local Storage
- 🔄 Data persists after page refresh
- 📱 Fully Responsive Design
- ⚛️ Built with React Functional Components

## 🚀 Getting Started

### Clone the Repository

```bash
git clone https://github.com/YourUsername/react-task-manager.git
```

### Navigate to the Project Folder

```bash
cd react-task-manager
```

### Install Dependencies

```bash
npm install
```

### Start Development Server

```bash
npm run dev
```

---

## 📖 How to Use

1. Enter a task in the input field.
2. Click the **ADD** button.
3. Click the checkbox to mark a task as completed.
4. Click the delete icon to remove a completed task.
5. All tasks are automatically saved in your browser.
